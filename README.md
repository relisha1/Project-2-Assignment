Assignment – Data Structures & Algorithms in C
This folder contains my solutions for the 5 programming questions. Each one is written in C and can be compiled and run on its own. I tried to keep the code straightforward and easy to follow.

How to Compile and Run
You will need a C compiler like gcc. Open a terminal, navigate to the folder where the files are, and use the commands below.
gcc q1_max_heap_FINAL.c -o q1 && ./q1
gcc q2_airport_heap_FINAL.c -o q2 && ./q2
gcc q3_kruskal_mst_FINAL.c -o q3 && ./q3
gcc q4_bfs_breach_FINAL.c -o q4 && ./q4
gcc q5_bellman_ford_FINAL.c -o q5 && ./q5
Only Q4 asks for user input. All others run automatically and print the results.
